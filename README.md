# Field of Junctions Initialization



## Requirements

Our code is implemented in pytorch. It has been tested using pytorch 1.6 but it should work for other pytorch 1.x versions. The following packages are required:

- python 3.x
- pytorch 1.x
- numpy >= 1.14.0
